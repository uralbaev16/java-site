JavaScript Yaratuvchisi

JavaScript dasturlash tilini Brendan Eich yaratgan. U 1995-yilda NetScape korporatsiyasida ishlab chiqarilgan va dastlab "Mocha" deb nomlangan.

Data Types (Ma'lumot Turlari)

  • String: Matnli qiymatlar. Misol: "Salom, dunyo!"
  • Number: Raqamli qiymatlar. Misol: 42 yoki 3.14
  • Boolean: Mantiqiy qiymatlar, faqat true yoki false qiymatlari bo'ladi.
  • Undefined: Qiymat hali belgilanmagan.
  • Null: Bo'sh yoki yo'q qiymat.
  • Object: Murakkab tuzilmalar, ma'lumotlarni birlashtirish uchun.

Math Objekti

Math obyekti matematik hisob-kitoblar uchun ko'plab foydali funksiyalarni o'z ichiga oladi.

  • Math.PI: Ï€ qiymati, ya'ni 3.14159...
  • Math.round(): Raqamni yaxlitlaydi. Misol: Math.round(4.7) natijasi 5 bo'ladi.
  • Math.sqrt(): Raqamning kvadrat ildizini topadi. Misol: Math.sqrt(16) natijasi 4 bo'ladi.
  • Math.random(): 0 va 1 orasida tasodifiy raqam qaytaradi.

If Statement

if operatori shartni tekshiradi va agar shart true bo'lsa, kod blokini bajaradi.

let yosh = 20;
if (yosh >= 18) {
  console.log("Siz kattasiz.");
} else {
  console.log("Siz voyaga yetmagansiz.");
}

Yuqoridagi misolda, agar yosh 18 yoki undan katta bo'lsa, "Siz kattasiz." xabari chiqadi, aks holda "Siz voyaga yetmagansiz." xabari chiqadi.
